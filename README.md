# interns-list
Recruitment task

## Clone repository

```sh
git clone https://github.com/DaveCzyz/interns-list.git
cd interns-list
```

## Install dependencies

```sh
npm install
```

### Start dev 

```sh
npm run dev
```

### Build production

```sh
npm run build
```

### Run Lint

```sh
npm run lint
```

### Run Prettier

```sh
npm run format
```
